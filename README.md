# The Trail AI Intelligence Project: Te Araroa

A professional website showcasing a 3,000km field research initiative to generate high-value datasets for conservation, trail management, and product innovation.

## About the Project

The Trail AI Intelligence Project transforms the Te Araroa trail into a 4-month-long outdoor laboratory, capturing data that is otherwise impossible to collect. As a Hiker-in-Residence and Field AI Researcher, this project turns an iconic 3,000km journey into a professional deployment that generates high-value datasets for multiple industries.

## Website Features

- **Modern, Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Professional Presentation**: Clean, modern interface suitable for potential sponsors
- **Three Partnership Tracks**: Detailed proposals for different types of organizations
- **Interactive Elements**: Smooth scrolling, hover effects, and mobile navigation
- **Contact Integration**: Direct email links and social media connections

## Deployment to GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com) and sign in
2. Click the "+" icon in the top right corner
3. Select "New repository"
4. Name your repository (e.g., `trail-ai-project`)
5. Make it public (required for free GitHub Pages)
6. Click "Create repository"

### Step 2: Upload Your Files
1. In your new repository, click "uploading an existing file"
2. Drag and drop all the website files:
   - `index.html`
   - `styles.css`
   - `script.js`
   - `PXL_20250809_194920538.jpg`
   - `README.md`
3. Add a commit message like "Initial website upload"
4. Click "Commit changes"

### Step 3: Enable GitHub Pages
1. Go to your repository's "Settings" tab
2. Scroll down to the "Pages" section
3. Under "Source", select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Click "Save"
6. Your site will be available at `https://yourusername.github.io/repository-name`

### Step 4: Custom Domain (Optional)
If you want to use a custom domain later:
1. In the Pages settings, enter your domain name
2. Add a CNAME file to your repository with your domain
3. Configure DNS settings with your domain provider

## Customization

### Updating Content
- Edit `index.html` to change text content
- Modify `styles.css` to change colors, fonts, or layout
- Update `script.js` to add new interactive features

### Adding PDF Downloads
To enable the download buttons:
1. Add your PDF files to the repository
2. Update the `href` attributes in `index.html` to point to your PDF files
3. Remove the JavaScript alert in `script.js`

### Changing Colors
The main color scheme uses blue (`#3182ce`). To change it:
1. Search for `#3182ce` in `styles.css`
2. Replace with your preferred color
3. Update related colors for consistency
